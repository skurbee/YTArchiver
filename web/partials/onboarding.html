Welcome to YTArchiver

Let's get you set up — this only takes a minute.

  • Choose where your archive lives

  • Optionally set up AI transcription (Whisper)

Pick an archive folder

YTArchiver saves each subscribed channel as a subfolder here. Choose an empty folder (or an existing YTArchiver archive). You can change it later in Settings.

Install dependencies

Download tools required
AI transcription optional
Sets up Python 3.11 + faster-whisper + torch. Large download (~3 GB with GPU acceleration) — it runs in the background and you can keep using the app. Safe to skip and set up later; downloading videos works without it.

You're all set

YTArchiver is ready. Add channels on the Subs tab, or paste a video URL on the Download tab to grab a single video.
